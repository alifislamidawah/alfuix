Introduction

ALUIX is a futuristic, modular SCSS + JS UI framework built to extend Bootstrap with advanced utilities, animations, and clipping effects for next-generation web design.

$ npm install aluix

Installation

You can install ALUIX via npm or use the CDN for quick setup:

$ npm install aluix
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/aluix/dist/css/aluix.min.css">

Folder Structure

Here’s how ALUIX organizes its SCSS and JS modules:

ALUIX/
├── scss/
│   ├── components/
│   ├── utilities/
│   ├── animations/
│   ├── sections/
│   └── aluix.scss
├── js/
│   ├── animations.js
│   ├── utilities.js
│   └── aluix.js
└── dist/
    ├── css/
    └── js/

Utilities

ALUIX provides a growing set of SCSS utilities beyond Bootstrap’s defaults — for sizing, filters, clipping, and transforms.

  • .w-10, .w-25, .w-50 — width utilities
  • .clip-triangle, .clip-hexagon — clip-path utilities
  • .filter-blur, .filter-glow — futuristic effects

Animations

Built-in motion utilities let you animate elements with ease:

<div class="reveal-up delay-2">Animated content</div>

Available effects: reveal-up, reveal-left, fade-in, float.

Plugins

ALUIX supports optional integrations for particles.js, fslightbox, and slick-carousel.

JavaScript

The JS layer handles motion triggers, UI states, and theme transitions:

ALUIX.init({
  motion: true,
  themeSwitch: true,
});

Customization

Modify SCSS variables in _variables.scss to change colors, animations, or border radii. ALUIX uses a modular structure for easy overrides.
