Download ALUIX

ALUIX is available via npm, CDN, or downloadable source files. Choose the method that best fits your workflow and start building futuristic interfaces faster.

1. Install via NPM

Install the latest ALUIX release using npm:

$ npm install aluix

Then import it into your SCSS and JS entry files:

// main.scss
@use 'aluix/scss/aluix';

// main.js
import 'aluix/dist/js/aluix.min.js';

2. Use CDN

Include ALUIX and Bootstrap directly in your HTML file via CDN:

<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css">
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/aluix/dist/css/aluix.min.css">

<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/js/bootstrap.bundle.min.js"></script>
<script src="https://cdn.jsdelivr.net/npm/aluix/dist/js/aluix.min.js"></script>

5. Build Tools

ALUIX uses sass and npm scripts for compilation and packaging:

$ npm run build     # Compile SCSS and bundle JS
$ npm run watch     # Watch for changes during development

You can extend the configuration to include autoprefixing or minification as needed.

6. Browser Support

ALUIX supports all modern browsers that are compatible with Bootstrap 5:

  • ✅ Chrome (latest 2 versions)
  • ✅ Firefox (latest 2 versions)
  • ✅ Safari (latest 2 versions)
  • ✅ Edge (latest 2 versions)

IE 11 and legacy browsers are not supported.
