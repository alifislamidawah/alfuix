Get Started with ALUIX

ALUIX is a modular SCSS + JS framework built on top of Bootstrap — designed for futuristic web design, with enhanced utilities, animations, and visual effects.

1. Install via NPM

To install ALUIX with npm, run the following command in your project directory:

$ npm install aluix

Then import ALUIX SCSS and JS into your project entry files:

// main.scss
@use 'aluix/scss/aluix';

// main.js
import 'aluix/dist/js/aluix.min.js';

2. Include via CDN

You can also include ALUIX directly from a CDN for quick prototyping:

<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css">
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/aluix/dist/css/aluix.min.css">

<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/js/bootstrap.bundle.min.js"></script>
<script src="https://cdn.jsdelivr.net/npm/aluix/dist/js/aluix.min.js"></script>

That’s all! You can now use ALUIX utilities, animations, and motion classes in your HTML.

3. Compile SCSS

ALUIX includes modular SCSS files you can customize and compile using the sass CLI:

$ sass src/scss:dist/css --style=compressed

Modify _variables.scss to adjust theme colors, glow effects, or animation speeds.

4. JavaScript Integration

ALUIX JS provides futuristic motion, theme switching, and plugin support.

// Initialize ALUIX
ALUIX.init({
  motion: true,
  reveal: true,
  themeSwitch: true
});

You can disable motion or customize animation timing through JS or SCSS variables.

5. Project Structure Example

Here’s a recommended setup for a new ALUIX project:

my-project/
├── src/
│   ├── scss/
│   │   ├── _variables.scss
│   │   └── main.scss
│   ├── js/
│   │   └── main.js
│   └── index.html
├── dist/
│   ├── css/
│   └── js/
└── package.json
