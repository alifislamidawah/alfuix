Installation with Vite

This guide shows you how to set up ALUIX in a modern project using Vite. It’s the fastest way to start building futuristic UIs using Bootstrap + ALUIX.

1. Create a Vite Project

Start by creating a new Vite project using npm:

$ npm create vite@latest my-aluix-app
$ cd my-aluix-app
$ npm install

Choose your preferred framework (for example, Vanilla or React).

2. Install ALUIX & Dependencies

Install ALUIX and Bootstrap via npm:

$ npm install aluix bootstrap sass

3. Import ALUIX Files

Inside your main.js (or main.tsx), import the ALUIX and Bootstrap JS bundles:

// main.js
import 'bootstrap/dist/js/bootstrap.bundle.min.js';
import 'aluix/dist/js/aluix.min.js';
import './style.scss';

Then, create a style.scss file in your src/ folder and import ALUIX SCSS:

// style.scss
@use "bootstrap/scss/bootstrap";
@use "aluix/scss/aluix";

4. Configure SCSS (Optional)

You can override ALUIX variables by adding a _variables.scss file before imports:

// style.scss
@use "variables"; // your overrides
@use "bootstrap/scss/bootstrap";
@use "aluix/scss/aluix";

Example variables:

$aluix-primary: #00f2ff;
$aluix-glow-strength: 0.8;

5. Run the Development Server

Start your dev environment:

$ npm run dev

Your ALUIX-powered app will be running at http://localhost:5173.

6. Build for Production

When you’re ready to deploy, build optimized assets with:

$ npm run build

This will generate minified CSS and JS files in the dist/ folder, including ALUIX assets.
